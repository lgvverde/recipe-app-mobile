# RecipeApp
- A simple Android application to get some random recipes for your daily cooking.


### Techs applied:
- [x] Coroutines;
- [x] Repository pattern;
- [x] Observer;
- [x] LiveData;
- [x] Retrofit;
- [x] Room;
- [x] MVVM;
- [x] Dependency Injection;
- [x] RecyclerView;
- [ ] Endless RecyclerView.
